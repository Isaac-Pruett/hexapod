# Hexapod
---
Work in Progress.

A project using a RP2040 board (Pimoroni Servo2040) to drive 18 servos in Embedded Rust using the [Embassy](https://embassy.dev) library.

Roadmap:
- [x] LED support
- [x] Drive servos
- [x] Servo array with PWM and PIO slices both used
- [ ] Add serial communication to Raspberry Pi
- [ ] Train RL policy for Raspberry Pi
- [ ] Add BROS to this project
